How to Play Tic-Tac-Toe

Steps:

  1. Open 2 different tabs and go to the following website http://localhost:3000/
  2. The player hosting the game should click host game this will create an access code
  3. The player joining the game should enter the access into the find game
  4. The host will get to move first followed by the player who joined.

Step 1: Open the Game

  • Host Player: Open your web browser and go to the game's website.
  • Joining Player: Do the same on a separate device or browser tab.

Step 2: Start a New Game

Host Player:

  • Click on the “Host Game” button as seen in Figure 4.0 below. This will create a unique access code for your game session.
  • Share this code with the player you want to invite.

Figure 4.0: Highlighting Host Game

Joining Player:

  • Enter the access code received from the host into the “Find Game” field.
  • Press enter or click the button to join the game as seen in Figure 4.1 below.

Figure 4.1: Highlighting Access Code Entry

Step 3: Playing the Game

  • The game board consists of a 3x3 grid.
  • The host player will make the first move, placing their symbol (X) in one of the grid squares by clicking on it.
  • Players take turns placing their symbols on the board.
  • The objective is to get three of your symbols in a row - horizontally, vertically, or diagonally.

Step 4: Winning the Game

  • The game ends when one player has three of their symbols in a row or all nine squares are filled without a winner.
  • If a player wins, the game will display a victory message.
  • If the board is filled and no player has won, the game will declare a draw.

Step 5: Starting a New Game

  • After a game concludes, players can start a new game by both players clicking on “Reset” as seen in Figure 4.2 below and repeating steps 2-4.

Figure 4.2: Highlighting Reset Button
